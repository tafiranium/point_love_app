### **VirtualPos утилита, автоматическое решение для упрощения рабочих процессов.**

**Две версии** (*offline developer / online client*).

*Быстрый старт: установка оффлайн-версии через Load unpacked в Chrome. Используется старшими и разработчиками для отладки приложения.*

*Стабильные версии: установка онлайн-версии через Load unpacked в Chrome специальный клиент доступен в группе автоматического VirtualPos. Версии отлаженные разработчиком, настроенные старшими продавцами, одобренные управляющими, готовы к использованию в кустах после бета-тестирования. Обновление и настройка данного приложения происходит налету.*

- developer-guide.md (инструкция для разработчика)
