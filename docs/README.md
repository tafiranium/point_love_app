## VirtualPos утилита

**Автоматическое решение для упрощения рабочих процессов.**

---

### 📦 Две версии клиента

- **offline** 
- **online**

---

### 🚀 Быстрый старт (оффлайн-версия)

- Установка через «Load unpacked» в Chrome  
- Используется **старшими и разработчиками** для отладки приложения

---

### ✅ Стабильные версии (онлайн-версия)

- Установка через «Load unpacked» в Chrome  
- Специальный клиент доступен в группе **автоматического VirtualPos**  
- Версии:
  - отлажены разработчиком
  - настроены старшими продавцами
  - одобрены управляющими
- Готовы к использованию **в кустах** после бета-тестирования  
- 🔄 Обновление и настройка приложения происходят **«налёту»**
